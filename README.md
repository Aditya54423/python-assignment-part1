# Part 1: Python Basics & Control Flow

## Project Overview
This repository contains a Python-based **Grade Tracker** system. It demonstrates the use of core Python concepts including variable management, data types, and conditional control flow to process student information.

## Features
* **Input Validation:** Ensures user inputs for grades are within a valid numerical range.
* **Conditional Logic:** Automatically calculates letter grades based on percentage thresholds.
* **Looping:** Allows for batch processing of multiple student records in a single session.
